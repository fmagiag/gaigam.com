Resume Start

  • Game Web - Mobile / Desktop: Avengers Gillette

    The project has the purpose of simulating a game in real time, where the player uses his smartphone as control of the characters that are shown on another screen.
    Technologies: html5, javascript, CSS3, createjs, nodejs, socket.io, acelerometer.

    Demo Youtube

  • Game Web - Mobile: Bradesco Bank

    Campaign developed like a game for Banco Bradesco, in order to publicize the various insurance options offered.
    Technologies: html5, javascript, CSS3, createjs, acelerometer.

    Demo Youtube

  • Android App: Connectt

    An application for condominium management. With this application, the resident can communicate the arrival to the concierge, access to news, messages of the receiver and classified of neighbors, discounts and promotions of contracted companies and service providers, consultations and reservations of dates in the common spaces, notify the arrival of visitors, goods and service providers, event management and arrival of guests.

    Play Store Demo Youtube

  • Android App: Vivo

    An application to the Vivo operator service management, with the App you can track your internet use, check the balance recharge, the validity of their claims and extract information about plan benefits, refer to the 2nd copy of your account, activate data packages, consult, renew and activate promotions, ask questions with the virtual assistant, activate international and contract packages to arrange in-store service.

    Play Store

  • Landing Page: CNA

    Landing Page developed for the CNA languages group with the intention of making the campaign's music available for download, provide the route to the nearest unit from the user's location and capturing more interested users through the registration form.
    Technologies: html5, javascript, CSS3, PHP, MySQL, GPS.

    Open Website

  • Landing Page: Spazio Felicitá

    Landing Page developed for the construction company Spazio Felicitá with the purpose of publicizing the venture, making the route available to the nearest unit from the user's location and scheduling a visit through a registration form.
    Technologies: html5, javascript, CSS3, PHP, MySQL, GPS.

    Open Website

  • Banner Rich Media: Youtube

    Banner Rich Media developed for YouTube to publicize the carnival campaign where the user jumped holding his smartphone and through the accelerometer he would be directed to the link of the event broadcast.
    Technologies: html5, javascript, CSS3, acelerometer.

    Demo Youtube

  • Banner Rich Media: Gillette

    Banner Rich Media developed for Gillette with the purpose of publicizing the campaign where it passes the idea of not having a place that prevents the power of your product, passing a little animation through four climatic conditions.
    Technologies: html5, javascript, CSS3.

    Demo Youtube

  • React: Project

    Project developed in the first module of the udacity nanodegree react, aims to manage books from a library, marking the books you are reading, you would like to read and the ones you have read.

    Open Website Github Code

  • IOT Project: SmartCall

    Project developed in the Java course of SENAI and awarded first in the state innovation program, where the project has the function of controlling equipment of a residence through software developed in Java and a hardware built to receive the DTMF signal and activate some function for residence control.

    Demo Youtube
